### **Étapes d’Installation**
### **1. Clonez le dépôt :**
   ```bash
   git clone https://github.com/votre-utilisateur/GAN_Mastery.git
   cd votre-repo
  ```

### **2. Installez les dépendances :**
```bash
pip install -r requirements.txt
````

### **3. Lancez le serveur :**
```bash
uvicorn app.main:app --reload
```
